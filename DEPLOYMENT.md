# Deployment Checklist

- Upload `index.html`, `.nojekyll`, `README.md`, `VERSION.txt`, and `CHANGELOG.md` to the repository root.
- Confirm GitHub Pages is enabled from `main` and `/ (root)`.
- Open the published URL in Safari and Chrome.
- Confirm the Saint Luke's demo loads.
- Test navigation, facility search, unit selection, room audit entry, Review Audit, History, and Posey Protect Report.
- Keep a backup copy of the ZIP package before replacing a live version.
