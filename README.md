# Posey Protect Purposeful Rounding — GitHub Pages Package

## Upload to GitHub

1. Open your GitHub repository.
2. Upload every file and folder from this package to the repository root.
3. Commit the changes.
4. Open **Settings → Pages**.
5. Select **Deploy from a branch**.
6. Choose **main** and **/(root)**, then save.
7. Wait for GitHub to display the live Pages URL.

## Install on iPhone or iPad

1. Open the GitHub Pages URL in Safari.
2. Tap **Share**.
3. Tap **Add to Home Screen**.
4. Name it **Posey Protect** and tap **Add**.

## Updating later

Replace `index.html` and change the cache name in `service-worker.js` (for example, v2 instead of v1) so Safari receives the newest version.

## Important demo note

This GitHub Pages build stores working data in the browser on that device. A public GitHub Pages repository should contain demonstration data only, not protected health information, confidential customer data, or internal pricing that is not approved for public hosting.
